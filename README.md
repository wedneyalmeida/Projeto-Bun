# Projeto-Bun
# Amazon Scraper com Bun + Vite

## 📦 Backend (Bun)
### Requisitos
- Bun instalado: https://bun.sh

### Instalação e execução
```bash
bun install
bun index.ts

🖥️ Frontend (Vite)
Requisitos
Node.js

Instalação e execução

cd frontend
npm install
npm run dev

🧪 Uso
Acesse http://localhost:5173

Digite a palavra-chave e clique em “Scrape”

Os produtos da primeira página da Amazon aparecerão

⚠️ Observações
O Amazon pode mudar a estrutura do HTML, quebrando o scraper

Este projeto é apenas para fins educacionais
